XHTML is a combination of HTML and XML

Good to know

  1. First step: add !DOCTYPE as the first line of the page!!!
  2. Second step: add xmlns attribute to the openint html tag! xmlns attribute specifies an xml namespace!
  3. Third step: ensure that all elements are properly closed and nested!
  4. Fourth step: change all attribute and elemnt name to lowercase!
  5. Fifth step: use id attribute instead of name attribute!
  6. Sixth step: make sure that all attribute values have quotes around them!
